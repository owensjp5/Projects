Basic Bioinformatics Pipeline Implementation

Usage
./run.sh output_directory gene1 gene2 

    Example: ./run.sh out_pipeline GSM801364 GSM801370

load.py
    Reads expression data from text files for given set of genes into CSV

compare.py
    Determines cell lines with most similar expression data within given set of genes
